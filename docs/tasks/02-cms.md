## משימות
- [ ] ליצור פרויקט Sanity (prod/dev datasets).
- [ ] להגדיר Schemas: Post, Page, Tag, Category, Author, Media.

- [ ] להוסיף דוגמאות תוכן (5 פוסטים, תגיות/קטגוריות, עמוד About).
